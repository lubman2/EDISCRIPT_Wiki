# PRINTPREVIEW

opens "Print Preview" window and shows prepared print. From the window is possible
        to see all pages, zoom pages or print.

### Format:

## PRINTPREVIEW

no parameters

### Example:

PRINTOPEN

        ...

        PRINTPREVIEW

        PRINTCLOSE
