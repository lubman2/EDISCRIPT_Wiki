# EXCELSAVE

saves data from opened excel file.

### Format:

## EXCELSAVE

no parameters

### Example:

EXCELOPEN "c:\mydir\test.xls"

        EXCELSET "A1" "Test"

        EXCELSAVE

        EXCELCLOSE
