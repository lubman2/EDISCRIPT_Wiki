# Excel Functions

work with *.xls and *.xlsx files. **EXCELOPEN** must be first function. Use long file name. Use **EXCELCLOSE** in the end of work.
        Do not forget to save your changes with **EXCELSAVE**.

EXCELOPEN opens an excel file EXCELGET gets value from excel cell EXCELSET puts a value in an excel cell EXCELSHEET selects excel sheet EXCELSAVE saves data in excel file EXCELCLOSE closes an opened excel file
