# File Functions

enable to work with files.

PROGPATH path of program INIREAD reads from an .INI file INIWRITE write in .INI file FINDPATH set path for finding files FINDFILE find file in specified path FILECOPY copies a file FILEEXIST test if file exist RENAME renames a file FILEEXT changes file name extension FILELENGTH retrieves length of file FILENAME gets a file name from the path file name FILEDELETE deletes a file FILEOPEN opens file for reading READLINE read one line form file READSEG read one segment from file READTEXT read specified number of characters from file FILECLOSE closes file FILETIME gets a file time FILEUNIQUE creates an unique file name GUID creates an unique file name WRITE writes a text in a file WRITEN writes a text in a file with a new line in the end CONVFILE converts file according code table
