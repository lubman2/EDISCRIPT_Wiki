# XML Functions

enable to work with XML files. **XMLOPEN** must be used for opening XML file. **XMLCLOSE**
        must be in the edn of work with XML files. Do not forget to save changes with **XMLSAVE**.

XMLOPEN opens XML document XMLERROR retrieves error text after XMLOPEN XMLROOTTAG retrieves root tag of  a XML document XMLTAG searchs for specified tag in XML document XMLTAGSET creates new tag XMLTEXTGET retrieves text of specified tag of XML document XMLTEXTSET writes text in a tag XMLATTRGET retrieves an attribute from XML tag XMLATTRSET sets an attribute in a XML tag XMLSAVE saves XML document in a file XMLCLOSE closes XML document
