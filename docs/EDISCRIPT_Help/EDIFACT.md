# EDIFACT Functions

enable to work with EDI files, converts data, read and write in EDI messages.

EDIREAD reads an EDI file EDICOUNT counts the EDI messages in file EDIMESSAGE sets on a message in the file EDICREATE creates EDI file EDIERROR get last a error text EDIEXPORT exports from EDI message EDIIMPORT imports data to an EDI file EDIIMPEDI imports an EDI file to an EDI file EDICHECK checks message syntax and mandatory fields in EDI message EDIFATHER sets father of EDI field EDIGET reads item from a message EDISET sets one item in EDI message EDIMSGLEN returns number of bytes in messages EDIREADMSG read one message from file EDIUNA adds or removes UNA segment EDIUNB adds or removes UNB segment EDIUNBMSG sets on a specified UNB segment EDIWRITE writes in an EDI file EDIRESET resets all variables EDIGETSEG retrieves segment and level for a specified gegment in message EDIGETSEGS retrieves number of segments COUNTER reads counter number EAN check EAN
