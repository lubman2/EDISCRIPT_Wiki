# String Functions

enable to work with text and time.

STRCOPY copies a string STRLEN returns length of string STRFIND returns the string found STRFINDX returns the repeated string found TIME gets actual time STRTOTIME converts a string to a time TIMETOSTR convert a time to a string TRIM trim string ~TO$ converts a floating point variable to a string STRUPPER converts string to upper case CONVTEXT converts text according code table GETFIELD get EDI field from string GETITEM get item from string GETTEXT get text from string
