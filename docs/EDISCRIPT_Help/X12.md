# ANSI X12 Functions

enable to work with ANSI X12 files, converts data, read and write in EDI messages.

X12READ reads an EDI file X12COUNT counts the X12 messages in file X12MESSAGE sets on a message in the file X12ERROR get last a error text X12EXPORT exports from X12 message X12IMPORT imports data to an X12 file X12CHECK checks message syntax and mandatory fields in X12 message X12FATHER sets father of X12 field X12GET reads item from a message X12SET sets one item in X12 message X12WRITE writes in an X12 file X12RESET resets all variables
