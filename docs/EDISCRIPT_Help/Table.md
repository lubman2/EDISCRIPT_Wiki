# Table Functions

enable to work with DBF tables.

TABLEOPEN opens a table TABLEOPENR opens a table for reading only TABLEOPENX opens a table without index TABLEOPENXR opens a table without index for reading only TABLEAPPEND appends a record in a table TABLECREATE creates a table TABLEDELETE deletes an actual record from table TABLEFIND searches in a table TABLENEXT repeatedly finds the next value TABLEREAD reads a field from a table TABLERECORD goes to a record in a table TABLEUPDATE updates a record in a table TABLEWRITE writes a field in an actual record TABLECLOSE closes a table
