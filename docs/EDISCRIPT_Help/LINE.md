# LINE

draws line. The pen to use to draw can be specified with the PEN function. Default
        pen is black.

### Format:

## LINE l t r b

l left coordinate of line t top coordinate of line r right coordinate of line b bottom coordinate of line

### Example:

PRINTOPEN

        LINE 10 10 200 100

        PRINTPREVIEW

        PRINTCLOSE
