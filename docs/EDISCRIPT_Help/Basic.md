# Basic Functions

Every script must start with **ACTION** and must be terminated with **END**
        or **ERROR**. Use **LET** for basic instruction.

; comment ACTION the first instruction of process END end of action ERROR end of action with an error GOTO goes to the line LABEL label for jump LET equation NEWACTION jump to other ACTION BEEP beep CALL calls procedure PROC beginning of a procedure RETURN returns from a procedure CONST text constant WAIT temporary end
