# EDISCRIPT Wiki

Vítejte v online dokumentaci EDISCRIPT systému!

Tato wiki obsahuje komplexní dokumentaci pro:

- **COMDIS** - Kompletní průvodce, vizuální přehledy, modelování a XML schéma
- **PCS** - Syntax guide, export a referenční příručka operací

## Rychlý start

Začněte s [COMDIS Kompletní průvodcem](COMDIS/COMDIS_KOMPLETNI_PRUVODCE.md) nebo podívejte se na [Vizuální přehled](COMDIS/COMDIS_VIZUALNI_PREHLED.md).

## Struktura dokumentace

```
docs/
├── COMDIS/     - Dokumentace COMDIS systému
└── PCS/        - Dokumentace PCS syntaxe
```

Pomocí navigace na vrchu najdete všechny dostupné dokumenty.
