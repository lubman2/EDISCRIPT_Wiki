# EDISCRIPT Wiki

Vítejte v online dokumentaci EDISCRIPT systému!

Tato wiki obsahuje komplexní dokumentaci pro:

- **PCS** - Skriptování, syntax guide a referenční příručka operací
- **COMDIS** - Podružná agenda: průvodce, vizuální přehledy, modelování a XML schéma

## Rychlý start

Začněte s [PCS Syntax guide](PCS/PCS_SYNTAX_GUIDE.md) nebo pokračujte na [Operace - referenční průvodce](PCS/PCS_OPERACE_REFERENCNI_PRUVODCE.md).

## PCS skriptování (hlavní)

PCS je jádro tohoto wiki. Najdete zde praktické návody a referenční materiály pro tvorbu a údržbu skriptů:

- [PCS Syntax guide](PCS/PCS_SYNTAX_GUIDE.md) - přehled syntaxe, pravidel a běžných vzorů
- [Operace - referenční průvodce](PCS/PCS_OPERACE_REFERENCNI_PRUVODCE.md) - kompletní popis operací a parametrů

## COMDIS (podružná agenda)

COMDIS slouží jako doplňková dokumentace k modelování a popisu dokumentů:

- [COMDIS Kompletní průvodce](COMDIS/COMDIS_KOMPLETNI_PRUVODCE.md)
- [COMDIS Vizuální přehled](COMDIS/COMDIS_VIZUALNI_PREHLED.md)
- [Modelování dokumentů](COMDIS/COMDIS_E2X_MODELCH_DOKUMENTACE.md)
- [XML schéma](COMDIS/COMDIS_XML_SCHEMAT.md)

## Doporučený postup

1. Projít [PCS Syntax guide](PCS/PCS_SYNTAX_GUIDE.md) a vyjasnit si pravidla zápisu.
2. Při implementaci opakovaně sahat do [Operace - referenčního průvodce](PCS/PCS_OPERACE_REFERENCNI_PRUVODCE.md).
3. Pokud potřebujete popis struktur dokumentů, využít COMDIS sekci výše.
